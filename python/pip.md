# pip

## 获取帮助
```
pip --help
```

## 升级 pip
```
pip install -U pip
```

## 安装包
```
pip install SomePackage
```

## 卸载包 
```
pip uninstall SomePackage
```

## 升级指定的包
```
pip install -U SomePackage
```

## 搜索包
```
pip search SomePackage
```

## 查看指定包的详细信息
```
pip show -f SomePackage
```

## 列出已安装的包
```
pip freeze or pip list
```

## 查看可升级的包
```
pip list -o
```
